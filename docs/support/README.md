## What is OnCallShift?

OnCallShift is a production incident management platform that helps engineering teams:

- **Respond faster** - Real-time alerts via push, SMS, email, and Slack
- **Coordinate effectively** - Escalation policies, on-call schedules, and team management
- **Resolve smarter** - AI-powered diagnosis, runbook automation, and cloud investigation
- **Learn continuously** - Postmortems, analytics, and performance reports
